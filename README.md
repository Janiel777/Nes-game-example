

# How to download and run the game on the emulator


## 1. Dirigirse a la seccion donde estan los releases
![image](https://github.com/Janiel777/Nes-game-example/assets/95184925/011a0a3b-641a-46cb-bb2f-bb16298c82d3)


## 2. Go to the section where the releases are
![image](https://github.com/Janiel777/Nes-game-example/assets/95184925/54f6dbe2-f830-48a3-8e01-cd23ff019988)


## 3. Open the workSpace file with an IDE. (VSCode in this case)
![image](https://github.com/Janiel777/Nes-game-example/assets/95184925/fa0b3cf8-8c96-4cc8-9a00-deb47e9d26cf)


## 4. Run run build task. You can do this by pressing Control+Shift+B
![image](https://github.com/Janiel777/Nes-game-example/assets/95184925/c0f302a5-43aa-4a14-b611-4063f6c3b5b8)

## 5. Open the emulator
![image](https://github.com/Janiel777/Nes-game-example/assets/95184925/56bd4b84-4e41-4116-a67d-c7b1256f9e83)
![image](https://github.com/Janiel777/Nes-game-example/assets/95184925/97e95197-f20a-45f7-8546-f54874afca97)

## 6. Press FILE<<OPEN and choose the demo.nes file
![image](https://github.com/Janiel777/Nes-game-example/assets/95184925/6e071c6e-ac7f-49c1-8376-676da816bebb)


## 7. You will be able to successfully play the game.
![image](https://github.com/Janiel777/Nes-game-example/assets/95184925/950ed301-851e-4431-ac87-929a5a4217b1)
